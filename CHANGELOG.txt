Teacher Value Prototype 322 - stable multi-file baseline

Changed from prototype 321:
- Split the single large HTML file into a more stable multi-file structure.
- index.html now handles the page structure and external library links.
- styles.css now contains the visual styling.
- districts-data.js now contains the district data.
- app.js now contains filtering, favorites, score guide, map, and UI behavior.
- Preserved the latest prototype 321 behavior/settings, including mobile filter fixes and the 60+ overall filter.

Recommended workflow:
- Edit styles.css for visual/layout changes.
- Edit app.js for behavior changes.
- Edit districts-data.js for district data changes.
- Keep this CHANGELOG updated after each prototype version.
